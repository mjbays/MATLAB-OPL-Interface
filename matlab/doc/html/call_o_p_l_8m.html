this matlab script loads the Java API for OPL, specifies the OPL model and data file, adds additional parameters to the data file, then returns the OPL solution as a MATLAB struct (soln) Author: Michael Yahknis (NREIP Internship program.

In this case, it is better to debug the model in the OPL IDE.
